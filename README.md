# News Insight Hub

A full-stack news aggregation and visualization dashboard. Fetches articles from external news APIs, stores them in a database, and displays trends and statistics through interactive charts.

## Tech Stack

**Backend:** Django 6.0, Django REST Framework, PostgreSQL 15, Python 3.x

**Frontend:** React 19, TypeScript, Vite, Tailwind CSS

## Prerequisites

- Python 3.8+
- Node.js 18+
- Docker & Docker Compose
